| 🔍 Model Observation       | What Model Sees       | 💬 Simple Explanation (For Normal People) |  Meaning                 |
| -------------------------- | ------------------------ | ----------------------------------------- | -------------------------- |
| Pulse amplitude low        | Weak signal peaks        | “Your blood vessels may be tight”         | Higher BP                  |
| Pulse amplitude high       | Strong peaks             | “Your blood vessels look relaxed”         | Lower BP                   |
| Sharp/narrow pulse         | Fast, thin waveform      | “Your pulse is sharp and tight”           | Higher BP                  |
| Wide pulse                 | Broad waveform           | “Your pulse is wide and smooth”           | Lower BP                   |
| Fast rise to peak          | Steep upstroke           | “Your pulse rises very quickly”           | Stiffer arteries / high BP |
| Strong second peak (notch) | Reflection wave present  | “Your pulse is bouncing back strongly”    | Increased BP               |
| Irregular peaks            | Uneven spacing           | “Your heartbeat is irregular”             | Stress / fatigue           |
| Fast peak intervals        | Short gaps between beats | “Your heart is beating fast”              | High HR                    |
| Slow peak intervals        | Long gaps between beats  | “Your heart is beating slowly”            | Low HR                     |
| Variable rhythm            | Changing intervals       | “Your heartbeat is not steady”            | HRV / stress               |
| Stable rhythm              | Consistent intervals     | “Your heartbeat is stable and calm”       | Relaxed state              |
| High-frequency activity    | Rapid variations         | “Your body shows signs of stress”         | Stress / arousal           |
| Smooth signal              | Low variation            | “Your body looks calm and relaxed”        | Rest state                 |
